# Loop Engineering: Reading List

Loop Engineering promises to be the new direction for orchestrating agents and
getting the most out of frontier models. A lot of smart people and orgs have
shared their take and guidelines on it, and I'll keep adding to this list as I
learn more.

- [@RLanceMartin](https://x.com/RLanceMartin) on creating self-correcting loops and how the Fable 5 model excels at this. [Post](https://x.com/RLanceMartin/status/2064397389189071163)
- [@sydneyrunkle](https://x.com/sydneyrunkle) on hill-climbing loops to rewrite the agent harness with improved configuration. [Post](https://x.com/sydneyrunkle/status/2066928783534289358)
- [@mem0ai](https://x.com/mem0ai) on the importance of memory for loop engineering. [Post](https://x.com/mem0ai/status/2067305118891163833)
- [@addyosmani](https://x.com/addyosmani) on the basic components of loop engineering. [Post](https://addyosmani.com/blog/loop-engineering/)
- [@Steve8708](https://x.com/Steve8708) on a visual planner to better comprehend the plan the agent creates. [Post](https://x.com/mem0ai/status/2067305118891163833)
