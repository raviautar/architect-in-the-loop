# teach

**Learning mode.** Layers on `pair` for when you want to understand a concept, not just get it done.

One idea: one concept per message, anchored to a real code snippet (never described in prose), with a single diagram extended in place instead of fresh walls of text, and a checkpoint before moving on. The moment you say "I'm lost," it backs up to the last thing that landed rather than pushing forward and piling on.

See [SKILL.md](SKILL.md).
