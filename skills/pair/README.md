# pair

**The voice.** The foundation skill every other one builds on: how the agent talks to you.

One idea: your attention is the scarcest thing in the loop, so every reply leads with the conclusion, stays one sentence by default, and carries zero AI tells and zero padding. Around that sit the rules that make an agent trustworthy to work with: be honest about what you didn't verify, recalibrate hard the moment you say "less text," mirror your own phrasing, and never write a comment the code already explains.

`teach` and `goal-alignment` inherit this skill. See [SKILL.md](SKILL.md) for the full cut list.
