# plain-english

**Sticky plain-English reset.** Layers on `pair`.

One idea: a one-off "less text" only fixes the next reply, then the agent drifts back. This skill turns that correction into a mode, say "less text" / "too verbose" / "plain english" once and it re-loads `pair`'s cut list and caps every reply at a few words of plain, read-aloud English until you lift it. The cap is counted in words, not sentences, so it can't be gamed by one long run-on.
